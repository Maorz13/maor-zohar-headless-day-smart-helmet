### Diagnosed causes

1. **Broken links → console errors on every page (review-bot gate):** the nav/footer link to `/cart` and `/contact`, which don't exist in the export (404). Also `GET edge.wixapis.com/ecom/v1/carts/current` 404s for visitors with no cart (called unconditionally on load).
2. **LCP 7.3–7.4s on shop/blog/bookings:** all content is client-fetched via the Wix SDK; the exported HTML has only skeletons, so the LCP element waits for JS + network.
3. **TBT 2,840ms home mobile:** `model-viewer.min.js` (936KB) + full SDK hydration on the main thread.
4. **~680–754 KiB unused JS per page:** `lib/wix.ts` creates ONE client importing every SDK module (stores, ecom, cart-v2, redirects, blog, bookings, data, forms) — every page bundles all of it.
5. **CLS 0.183 blog:** loading skeleton doesn't match final list dimensions.
6. **color-contrast failures on home + shop (A11y 96).**
